## ğŸ¨ CUSTOMIZATION GUIDE

### Change Brand Name
Find and replace "RemoteFlow" with your name in all files:
- Logo text in navigation
- Footer
- Meta titles

### Update Colors
In `styles.css`, find the `:root` section (line ~20):
```css
--primary: #667eea;     /* Main purple */
--secondary: #764ba2;   /* Dark purple */
--accent: #f59e0b;      /* Orange */
```

Change these hex codes to your brand colors!

### Update Products
In `products.html`:
1. Find `$47` and `$37` - replace with your prices
